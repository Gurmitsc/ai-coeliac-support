# Contributing

Thank you for your interest in contributing to this project.

This repository explores how artificial intelligence can responsibly support people living with coeliac disease by improving access to trusted information and reducing misinformation.

## Ways to Contribute

We welcome contributions from:

- developers
- researchers
- health professionals
- people living with coeliac disease
- charities and advocacy organisations

Possible contributions include:

- improving prompt design for responsible AI responses
- identifying trusted information sources
- proposing AI use cases that support the coeliac community
- highlighting misinformation trends about gluten or coeliac disease
- suggesting improvements to documentation

## Responsible AI Principles

Because this project deals with health-related information, contributors should follow a few key principles:

- AI systems should not replace medical professionals.
- Information should prioritise trusted sources such as NHS guidance and Coeliac UK.
- Responses should encourage users to seek medical advice where appropriate.
- Contributions should aim to reduce misinformation rather than amplify it.

## How to Contribute

1. Open an issue to propose an idea or improvement.
2. Submit a pull request with suggested changes.
3. Clearly explain the reasoning behind your contribution.

## Code of Conduct

Contributors are expected to engage respectfully and constructively with others in the community.

The goal of this project is to explore how AI can responsibly support people living with coeliac disease while maintaining trust and safety.
